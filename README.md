# Home_price_prediction using machine learning  algorithm 

## Overview

This project is aimed at developing a machine learning model to predict home prices based on various features such as square footage, number of bedrooms, number of bathrooms, and location. The project utilizes a dataset of historical home sales to train and evaluate the model.
